# Parte 4 — Listening to a News Item

## Contexto Típico

- 1 reportagem de notícia (~1,5–2 min), **1 falante** em estilo formal de rádio/TV
- Temas canadenses locais: evento comunitário, anúncio municipal, incidente público, história de interesse humano, segurança pública
- **5 questões**, formato drop-down (4 opções cada), ~3–5 min para responder
- **Dificuldade:** Médio — entrega rápida e densa, sem pausa, sem repetição

Estilo jornalístico: voz passiva, vocabulário formal, frases compactas. A primeira frase é a mais importante — equivale ao lead da notícia e cobre o assunto principal.

---

## O que Mais Cai

- O tópico principal (quase sempre na primeira frase)
- Quem disse o quê — repórter, prefeito, críticos, moradores podem ter posições diferentes
- Números, estatísticas e percentuais no contexto certo
- Causa e efeito do evento
- O que acontece a seguir (declaração final prospectiva)

---

## Palavras-Gatilho para Prestar Atenção

| Tipo | Exemplos |
|------|----------|
| Fonte / atribuição | "According to...", "Officials say...", "Critics argue...", "Residents report...", "The mayor stated..." |
| Causa | "due to", "as a result of", "following", "in response to", "triggered by" |
| Efeito / resultado | "leading to", "resulting in", "which caused", "this has meant that" |
| Prospectivo (futuro) | "is expected to", "will be", "plans to", "is set to", "the next step is" |
| Contraste de posição | "while supporters say...", "however, critics...", "on the other hand" |

> **Atenção especial:** sempre que ouvir uma frase de atribuição ("According to X..."), anote quem é X e o que ele defende — isso é testado com frequência.

## O que Anotar no Papel

### Modelo — Framework 5W1H (monte ANTES do áudio)

```
QUEM:    ___________________________________
O QUÊ:   ___________________________________   ← 1ª frase = resposta quase certa
QUANDO:  ___________________________________
ONDE:    ___________________________________
POR QUÊ: ___________________________________
COMO:    ___________________________________

STATS:   ___________________________________   (todos os números com unidade)
FONTES:
  F1: ___________ → posição: _______________
  F2: ___________ → posição: _______________
  F3: ___________ → posição: _______________

ÚLT:     ___________________________________   (última frase / o que vem a seguir)
```

### Símbolos recomendados

| Símbolo | Significado |
|---------|-------------|
| ✓ | apoia / a favor |
| ✗ | contra / critica |
| ✓? | apoia com ressalva |
| → | causa leva a efeito |
| ↑ / ↓ | aumento / queda (com número) |
| % | percentual — sempre anote o contexto |
| $ | valor monetário |
| [F] | declaração final — circule |
| " " | citação direta — marque para lembrar que é da fonte, não do repórter |

### Exemplo de anotação real

```
QUEM:    Cidade de Vancouver, Depto de Parques
O QUÊ:   novo programa de horta comunitária
QUANDO:  a partir de 1º de abril
ONDE:    East Vancouver — 3 locais
POR QUÊ: reduzir insegurança alimentar (↑30% desde 2020)
COMO:    inscrições abertas no site da cidade

STATS:   $500K orçamento | 200 lotes disponíveis
FONTES:
  F1: Prefeito → ✓ "iniciativa necessária"
  F2: Moradores → ✓? "se estacionamento for resolvido"
  F3: Críticos → ✗ custo alto

ÚLT:     [F] "resultados esperados para o outono"
```

## Armadilhas e Erros Comuns

| Armadilha | Detalhe |
|-----------|---------|
| **Confusão de fonte** | Maior armadilha — quem disse o quê? O repórter narra, mas as posições pertencem às fontes |
| **Copiar palavras do áudio** | Respostas corretas usam sinônimos — opções com palavras idênticas ao áudio costumam ser distratores |
| **Números sem contexto** | Múltiplos números aparecem — anote sempre o contexto junto ($500K = orçamento, não prazo) |
| **Relaxar no final** | A última frase ("o que acontece a seguir") é testada com frequência — atenção até o fim |
| **Estilo formal desconhecido** | Se só praticou com conversas informais, o noticiário pode soar rápido — treine com CBC Radio |

---

## Estratégia de Resposta

1. **Concentração total na primeira frase** — ela define o tema e é quase sempre testada diretamente
2. Monte o 5W1H no papel antes do áudio começar — slots prontos para preencher
3. Ao ouvir uma frase de atribuição, **escreva a fonte + posição com símbolo** (✓ ou ✗) imediatamente
4. Anote todo número **com contexto** — nunca só o número
5. Com os menus drop-down: **pré-leia as opções antes do áudio** — elas revelam que tipo de dado cada questão vai pedir
6. Reserve o espaço "ÚLT" para a última frase — não relaxe antes do áudio terminar
