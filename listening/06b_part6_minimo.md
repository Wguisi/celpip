# Parte 6 — Mínimo Essencial para Anotar

> Versão simplificada. Quando o monólogo é rápido e parece muita coisa, foque só nisso.

---

## O único objetivo desta parte

Saber, ao final, **quem pensa o quê — e separar isso do que o narrador pensa**.

O narrador fala por ~3 minutos, mas está apresentando opiniões de outras pessoas. Sua tarefa é não confundir as opiniões relatadas com a opinião do próprio narrador.

---

## Template mínimo (desenhe no papel em 10 segundos)

```
VP1 [quem]:
VP2 [quem]:
NAR:
---
ÚLT:
```

Duas linhas para as perspectivas, uma para o narrador, uma para a última frase.

---

## O que escrever em cada linha

Para cada perspectiva, anote apenas:
- **quem** é (ex: "moradores", "planejadores", "críticos")
- **posição**: + a favor | - contra | ~ neutro/condicional
- **argumento principal** em 2–3 palavras (ex: "custo alto", "melhora tráfego")

Para o narrador:
- Só a **postura implícita** ao final — ele tende a a favor (+) ou contra (-)?
- Muitas vezes o narrador não declara — anote "?" se não ficou claro

```
VP1 [planejadores]: + / melhora mobilidade / dado: ↑40% uso
VP2 [comerciantes]: - / obra prejudica / dado: ↓35% receita
NAR: +? (parece apoiar mas com cautela)
---
ÚLT: "consultas adicionais no outono"
```

---

## O sinal mais importante do áudio

Quando ouvir **"however / on the other hand / in contrast / not everyone agrees"** → **mude de coluna no papel**.

Essas palavras marcam a fronteira entre uma perspectiva e a próxima. Se você pegar esses sinais, você não mistura as perspectivas.

---

## A armadilha número 1

O narrador diz: *"Critics believe the plan will fail."*

Isso **não significa** que o narrador acha que o plano vai falhar.

Sempre que ouvir "according to / critics argue / supporters believe / residents say" — o que vem depois é **opinião de outro**, não do narrador.

Marque com parênteses no papel para lembrar: `(críticos: plano vai falhar)`

---

## Símbolos — só os essenciais

| Símbolo | Quando usar |
|---------|-------------|
| + | a favor / apoia |
| - | contra / critica |
| ~ | neutro ou condicional |
| ( ) | opinião relatada — não é do narrador |
| ★ | ponto mais importante |
| ÚLT | última frase do monólogo |

---

## Exemplo real completo (em 5 linhas)

Áudio: debate sobre nova linha de trem no centro

```
VP1 [planejadores]: + / mobilidade / ↑40% uso
VP2 [comerciantes]: - / obra / ↓35% receita
NAR: +? "dados são sólidos"
---
ÚLT: ★ "consultas previstas para o outono"
```

Com isso você consegue responder:
- Quem apoia e por quê? Planejadores, dados de uso
- Quem é contra e por quê? Comerciantes, perda de receita
- O narrador tem posição? Parece apoiar, mas com cautela
- O que foi dito no final? Novas consultas no outono

---

## Quando o monólogo acabar

Antes de responder, pergunte a si mesmo rapidamente:
1. Quem é a **perspectiva 1** e qual o argumento principal?
2. Quem é a **perspectiva 2** e qual o argumento principal?
3. O **narrador** parece a favor, contra ou neutro?
4. O que foi dito na **última frase**?

Se conseguir responder as 4, está pronto para as 6 questões.
