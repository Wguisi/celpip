# Parte 4 — Reading for Viewpoints

## Contexto Típico

- Você lê um artigo de opinião (editorial, post, carta à comunidade) de 300–500 palavras com **2–3 perspectivas diferentes sobre um tema polêmico**
- Temas típicos: política de transporte, desenvolvimento urbano, meio ambiente vs. economia, educação, saúde pública
- Depois completa um comentário/resposta com 5 lacunas
- **10 questões** | ~13 minutos | **A parte mais difícil**

**Formato:**
- Q1–5: múltipla escolha sobre o artigo principal (quem acredita no quê, argumento principal, tom, inferência)
- Q6–10: fill-in-the-blank completando o comentário/resposta (dropdown)

---

## O que Mais Cai

- Qual é a posição de cada pessoa sobre o tema?
- Qual é o argumento principal de cada lado?
- O autor do artigo apoia qual visão — ou está só relatando?
- Tom e emoção de cada falante
- Técnicas de persuasão usadas (apelo a dados, apelo emocional, pergunta retórica)
- Preenchimento do comentário: o que encaixa mantendo coerência com a posição do respondedor?

---

## Estratégia Central: T-Chart de Perspectivas

**Monte isso no papel antes de ler o artigo** (use o título para saber o tema):

```
TEMA: _________________________________

LADO A [quem?]          LADO B [quem?]
-----------             -----------
posição: + / -          posição: + / -
argumento 1:            argumento 1:
argumento 2:            argumento 2:
tom: ________           tom: ________
evidência: S/E/P        evidência: S/E/P
```

**Códigos de evidência:**
- S = estatística / dado numérico
- E = opinião de especialista / autoridade
- P = argumento de política / princípio

Preencha o T-Chart enquanto lê. Isso torna todas as Q1–5 uma consulta rápida.

## Palavras-Gatilho para Marcar na Leitura

### Sinalizadores de opinião pessoal
| Expressão | O que indica |
|-----------|-------------|
| "I believe / I think / I feel / In my view" | posição direta do autor ou do falante |
| "should / must / ought to / have to" | recomendação forte |
| "unfortunately / thankfully / clearly / obviously" | julgamento de valor — revela atitude |
| "it is evident that / it is clear that" | o falante trata como fato, mas é opinião |

### Sinalizadores de opinião relatada (atenção — não é do narrador!)
| Expressão | O que indica |
|-----------|-------------|
| "Critics argue / Supporters believe / Residents say" | opinião de outro grupo, não do autor |
| "According to / As X puts it / X claims that" | atribuição — o que vem depois é de X, não do narrador |
| "Some people feel / Many argue / Others suggest" | posição de terceiros |

### Concessão (armadilha frequente)
| Expressão | O que indica |
|-----------|-------------|
| "While I understand / Although X has merit / I acknowledge that" | autor reconhece o outro lado, MAS não concorda |
| "Despite this / Even so / Nevertheless" | autor vai retomar a própria posição logo depois |

> **Regra de ouro:** "Critics argue the plan will fail" ≠ o autor acha que vai falhar. Frase de atribuição = opinião de outro.

## O que Anotar no Papel

```
TEMA: _________________________________

A [nome/grupo]:         B [nome/grupo]:
posição: + / -          posição: + / -
arg: __________         arg: __________
evidência: ____         evidência: ____
tom: __________         tom: __________

NARRADOR: relata / apoia A / apoia B / neutro?

RESPONDEDOR (Q6–10): apoia A / apoia B / neutro?
```

A última linha é crítica para as Q6–10: se o respondedor apoia o Lado A, todas as 5 lacunas devem refletir essa posição.

## Estratégia de Resposta (passo a passo)

1. **2 minutos iniciais:** leia o artigo para entender o debate — quem são os lados e o que cada um defende
2. **Monte o T-Chart** enquanto lê (não antes — use a leitura para preenchê-lo)
3. **Q1–5:** use o T-Chart como consulta — quase toda questão é "qual lado diz X?"
4. **Antes de Q6–10:** identifique a posição do respondedor (apoia A, B ou neutro?)
5. **Q6–10:** preencha cada lacuna com a opção que:
   - É coerente com a posição do respondedor
   - Tem o tom certo (não coloque linguagem negativa numa resposta entusiasmada)
   - É gramaticalmente correta no contexto da frase
   - **Não** descreve a posição oposta como se fosse a do respondedor

## Armadilhas e Erros Comuns

| Armadilha | Detalhe |
|-----------|---------|
| **Confusão de atribuição — erro #1** | Atribuir opinião de A para B, ou achar que o narrador concorda com o que está apenas relatando |
| **Rótulos parecidos** | "Volunteering coordinator" vs "community organizer" — o exame usa títulos similares para confundir. Rastreie por nome, não por título |
| **Resposta invertida** | Uma das opções erradas é exatamente o oposto da correta — descreve o que o lado contrário pensa |
| **Grammatically correct, wrong viewpoint** | Para fill-in-the-blank, encaixe gramatical não é suficiente — precisa também ser coerente com a posição do respondedor |
| **"Mencionado" ≠ "acreditado"** | O autor pode descrever uma visão que existe na sociedade sem pessoalmente defendê-la |
| **Linguagem absoluta** | Respostas com "always / all / everyone / never" quase sempre erradas se o texto usa "many / often / some" |

## Técnicas de Persuasão (podem ser testadas diretamente)

| Técnica | Exemplo no texto |
|---------|-----------------|
| Apelo emocional | "Is this really what we want for our children?" |
| Apelo à autoridade | "According to Dr. Smith's research..." |
| Estatística | "Studies show a 40% increase in..." |
| Pergunta retórica | "Can we really afford to ignore this?" |
| Descarte do oponente | "Critics claim X, but in reality..." |
| Linguagem extrema | "absolutely unacceptable", "completely irresponsible" |

---

## Símbolos para usar enquanto lê

| Símbolo | Quando usar |
|---------|-------------|
| A+ / A- | posição do lado A (a favor / contra) |
| B+ / B- | posição do lado B |
| ( ) | opinião relatada — não é do narrador |
| ⚠ | concessão — narrador cita o outro lado mas não concorda |
| ★ | argumento principal de um lado |
| S / E / P | tipo de evidência (estatística / expert / política) |
| ~ | neutro / condicional |
