## Expressoes para Evitar / Avoid These

| Evite | Use no lugar |
|-------|-------------|
| "It's a good thing" | "This is a beneficial measure" |
| "Things will be better" | "Conditions will improve significantly" |
| "People will be happier" | "Individuals will experience greater well-being" |
| "I think it's important" | "It is essential to consider that" |
| "This is very bad" | "This is detrimental to [specific group/outcome]" |
| "We should do this" | "This initiative warrants serious consideration" |
| "A lot of people" | "A significant proportion of the population" |
| "In today's world" | "In the current context / In contemporary society" |
| "As everyone knows" | "It is widely recognized that" |
| "I will talk about" | (just say it — avoid announcing what you'll do) |
