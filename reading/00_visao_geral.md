## Estratégias Universais (Valem para Todas as Partes)

### Abordagem de leitura

- **Nunca leia o texto inteiro antes de ver as questões** (exceto P3 onde você lê os tópicos dos parágrafos)
- Use leitura por escaneamento: extraia palavras-chave das questões e caçe no texto
- Respostas corretas **quase nunca** copiam o texto — usam sinônimos e paráfrases

### Regras de tempo

- **Regra dos 60 segundos:** se ficou mais de 60s em uma questão sem encontrar a resposta, chute e avance
- Nunca gaste mais de 2 min em uma questão — não há tempo sobrando nesta prova

### Armadilhas universais

| Armadilha | Como evitar |
|-----------|-------------|
| **Linguagem absoluta** | Respostas com "always / never / all / everyone" quase sempre erradas se o texto usa "often / some / many" |
| **Copiar palavras do texto** | Opções com frases idênticas ao texto são frequentemente distratores |
| **Conhecimento externo** | Responda só com o que está no texto — nunca com o que você "sabe" sobre o assunto |
| **Confusão de atribuição** | Em P4 especialmente: saber quem disse o quê é mais importante que entender o quê |

### Sinônimos para memorizar (respostas usam paráfrases)

| Palavra no texto | Pode aparecer como |
|-----------------|-------------------|
| increase / rise | grow / surge / expand / go up |
| cost / price | fee / expense / charge / rate |
| problem | challenge / difficulty / obstacle / issue |
| important | essential / vital / crucial / key |
| show | demonstrate / indicate / reveal / suggest |
| begin / start | launch / initiate / introduce / establish |
| use | apply / employ / utilize / implement |
