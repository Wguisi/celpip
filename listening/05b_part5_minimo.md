# Parte 5 — Mínimo Essencial para Anotar

> Versão simplificada. Quando o vídeo é rápido e você não consegue gerenciar tudo, foque só nisso.

---

## O único objetivo desta parte

Saber, ao final, **quem apoia, quem é contra, e o que foi decidido**.

Tudo que você anota deve servir para responder essas três perguntas.

---

## Template mínimo (desenhe no papel em 10 segundos)

```
A:
B:
C:
---
DEC:
```

É só isso. Três linhas para os falantes, uma linha para a decisão.

---

## O que escrever em cada linha

Para cada falante, anote apenas:
- **inicial ou apelido** (ex: "A = João/suéter azul")
- **posição com símbolo**: ✓ apoia | ✗ contra | ? incerto
- **se mudar de posição**: risque e escreva a nova (ex: ✗ → ✓)
- **a condição, se houver** (ex: ✓ se prazo 6m)

```
A: ✓
B: ✗ → ✓ se prazo 6m
C: ✗
---
DEC: piloto 6 meses ✓
```

Isso já é suficiente para responder a maioria das questões.

---

## O que ignorar quando estiver sobrecarregado

Se o vídeo estiver rápido demais, **largue as anotações e só escute** os momentos abaixo:

1. Quando alguém **muda de posição** ("actually... fair enough... okay I'll support it")
2. Quando o moderador **resume ou finaliza** ("so we're agreed...", "let's go with...")
3. Quando alguém coloca uma **condição** ("as long as...", "provided that...")

Esses três momentos cobrem a maioria das 8 questões.

---

## Símbolos — só os essenciais

| Símbolo | Quando usar |
|---------|-------------|
| ✓ | apoia |
| ✗ | contra |
| ? | não ficou claro |
| → | mudou para / condição |
| ★ | decisão final |

---

## Exemplo real completo (em 5 linhas)

Audio: reunião sobre expandir escritório

```
A (gerente): ✓
B (equipe):  ✗ → ✓ se 6 meses
C (finanças): ✗
---
DEC: ★ piloto 6m, revisar Jan
```

Com isso você consegue responder:
- Quem apoiou? A e B (com condição)
- Quem era contra? C
- O que foi decidido? Piloto de 6 meses
- O que fez B mudar? O prazo de 6 meses

---

## Quando o vídeo acabar

Antes de responder, pergunte a si mesmo rapidamente:
1. Quem era **a favor**?
2. Quem era **contra**?
3. O que foi **decidido**?
4. **Alguém mudou** de ideia? Por quê?

Se você conseguir responder as 4, está pronto para responder as 8 questões.
