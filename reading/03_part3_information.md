# Parte 3 — Reading for Information

## Contexto Típico

- Você lê um artigo informativo dividido em **4 parágrafos rotulados A, B, C, D** (300–500 palavras)
- Temas não especializados: história, ciência, natureza, questões sociais, geografia ou cultura canadense
- Tom: neutro, objetivo, sem forte opinião
- **9 questões** | ~10 minutos

**O formato é único:** a maioria das questões pergunta **em qual parágrafo (A, B, C ou D) está a informação**. A opção E significa "não mencionado em nenhum parágrafo".

---

## O que Mais Cai

- Em qual parágrafo o autor menciona X?
- Qual é o tema principal do parágrafo B?
- Recuperação de fato específico: quando aconteceu / quanto custa?
- Inferência: o que o autor implica sobre X?
- Vocabulário em contexto: o que "X" provavelmente significa aqui?
- Não mencionado (E): a informação não aparece em nenhum parágrafo

---

## Estratégia Central: Mapa de Parágrafos

**Antes de ver as questões**, leia apenas a primeira frase de cada parágrafo e monte isso no papel:

```
A = _________________________  (tema em 2–3 palavras)
B = _________________________
C = _________________________
D = _________________________
```

Exemplo:
```
A = introdução / visão geral
B = história / origem
C = benefícios / processo
D = desafios / futuro
```

Isso leva ~60 segundos e transforma cada questão em uma busca rápida, não uma releitura do texto inteiro.

---

## Palavras-Gatilho dentro dos Parágrafos

Essas palavras revelam a estrutura interna do parágrafo e ajudam a localizar sub-temas:

| Tipo | Exemplos |
|------|----------|
| Contraste | however, on the other hand, although, yet, despite, in contrast |
| Adição | furthermore, additionally, also, moreover, in addition |
| Causa/efeito | therefore, as a result, consequently, because, due to |
| Tempo/sequência | initially, subsequently, eventually, prior to, following |
| Exemplo | for instance, specifically, such as, in particular, including |

Quando você vê essas palavras durante o escaneamento, elas marcam onde sub-tópicos mudam dentro do parágrafo — útil quando a questão pede um detalhe específico, não o tema geral.

---

## O que Anotar no Papel

```
A = _____________  (tag de 2–3 palavras)
B = _____________
C = _____________
D = _____________

Keywords das questões:
Q1: ____________
Q2: ____________
Q3: ____________
(etc.)
```

Anote 1–2 palavras-chave de cada questão antes de escanear. Assim você sabe exatamente o que está caçando.

---

## Estratégia de Resposta (passo a passo)

1. **Leia a 1ª frase de cada parágrafo** e monte o mapa A/B/C/D (~60s)
2. **Leia TODAS as 9 questões** e extraia a palavra-chave de cada uma
3. **Escaneie parágrafo por parágrafo** — ao escanear A, identifique quais questões A pode responder. Faça o mesmo com B, C, D.
4. **Só commite respostas após ler os 4 parágrafos** — não decida antes de terminar
5. **"Não mencionado" (E) encontra-se por exclusão** — se varrreu A, B, C, D e não achou, então é E

---

## Regras Críticas

**Um parágrafo pode responder múltiplas questões** — não assuma que é 1 questão por parágrafo.

**Não trave em uma questão.** Se depois de 60s não achou, marque a melhor opção e avance — volte se sobrar tempo.

**Paráfrases são inevitáveis.** "The program was launched" no texto aparece como "the initiative began" na questão. Treine reconhecer sinônimos.

---

## Armadilhas e Erros Comuns

| Armadilha | Detalhe |
|-----------|---------|
| **Responder antes de ler todos os parágrafos** | O erro mais comum — você acha que é A, mas é C. Só commite depois de escanear tudo |
| **Buscar palavras idênticas** | As questões parafraseiam o texto de propósito. Busque o significado, não a forma |
| **"Mencionado" ≠ "tema central"** | A questão pode pedir onde um detalhe é citado de passagem, não onde é o assunto principal |
| **Eliminar parágrafo sem ler** | Nunca descarte um parágrafo como possível resposta antes de realmente escaneá-lo |
| **Confundir inferência com fato** | "O autor implica X" ≠ "O autor diz X" — inferência requer leitura de subtexto |

## Símbolos para usar no papel

| Símbolo | Quando usar |
|---------|-------------|
| A/B/C/D | tag de cada parágrafo — escreva ao lado do resumo |
| ★ | fato importante / estatística — quase sempre cai em questão |
| E | não mencionado em nenhum parágrafo |
| ~ | "parecido com / pode ser essa resposta" — marque para revisar |
| ✓ | resposta confirmada |
| ? | dúvida — volte se sobrar tempo |
