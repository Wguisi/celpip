# Parte 5 — Listening to a Discussion

## Contexto Típico

- 1 clipe de **vídeo** (~1,5–2,5 min) com **3 falantes**
- Cenários: reunião de trabalho, comitê comunitário, planejamento de evento, decisão em grupo
- **8 questões**, drop-down, ~4–9 min para responder
- **Dificuldade:** Médio a difícil — a única parte com vídeo; 3 opiniões simultâneas para rastrear

Os falantes podem ser: moderador + 2 participantes, ou 3 colegas tomando uma decisão juntos. As posições mudam ao longo da conversa — a posição final é o que importa.

---

## O que Mais Cai

- Posição final de cada falante (não a inicial)
- Quem disse o quê (atribuição de fala)
- Condição que fez alguém mudar de ideia
- O que foi decidido / o que ficou pendente
- Detalhes visuais: roupa, cena, quem estava presente

---

## Palavras-Gatilho para Prestar Atenção

| Tipo | Exemplos |
|------|----------|
| Concordância total | "Absolutely", "Exactly", "I couldn't agree more", "That works for me", "I'm on board" |
| Concordância condicional | "Could work if...", "As long as...", "Provided that...", "I'll support it if...", "Let's pilot it first" |
| Discordância | "I see your point, but...", "I'm not so sure", "I'm uneasy about...", "I don't think we can commit" |
| Mudança de posição | "Actually, you know what...", "Fair enough", "That changes things", "Okay, I'll come around" |
| Decisão final | "So we're agreed on...", "Let's go with...", "The plan is...", "We'll move forward with..." |
| Moderador encerrando | "So to summarize...", "Let's finalize...", "We'll recommend...", "Moving on..." |

> **Atenção:** "Could work" e "I support it" **não são equivalentes**. Concordância condicional ≠ concordância total. As questões testam exatamente essa diferença.

## O que Anotar no Papel

### Modelo — Quadrante com linha de evolução

```
┌─────────────────────────────┬─────────────────────────────┐
│  A: ___________ (papel)     │  B: ___________ (papel)     │
│  visual: ________________   │  visual: ________________   │
│  P0: ___________________    │  P0: ___________________    │
│  mudança: ↑ ____________    │  mudança: ↑ ____________    │
│  Pf: ___________________    │  Pf: ___________________    │
│  condição: _____________    │  condição: _____________    │
├─────────────────────────────┼─────────────────────────────┤
│  C: ___________ (papel)     │  ★ DECISÃO FINAL            │
│  visual: ________________   │  _________________________  │
│  P0: ___________________    │  PENDENTE / SEM ACORDO:     │
│  Pf: ___________________    │  _________________________  │
└─────────────────────────────┴─────────────────────────────┘
```

**Legenda dos campos:**
- P0 = posição inicial
- Pf = posição final
- visual = roupa ou característica visual (para questões visuais)

### Símbolos recomendados

| Símbolo | Significado |
|---------|-------------|
| ✓ | apoia |
| ✗ | opõe |
| ✓? | apoia com condição / hesitação |
| ✗→✓ | mudou de contra para a favor |
| ✓→✗ | mudou de a favor para contra |
| → | condição que causou a mudança |
| !! | emoção forte (entusiasmo ou irritação) |
| ? | posição incerta / não ficou claro |
| ~ | neutro / indiferente |
| [nv] | pista não-verbal (acenou, cruzou os braços, etc.) |

### Exemplo de anotação real

```
┌─────────────────────────────┬─────────────────────────────┐
│  A: gerente (azul)          │  B: equipe (cinza)          │
│  P0: expandir ✓             │  P0: ✗ rápido demais        │
│  mudança: preocupação $     │  mudança: → prazo 6 meses   │
│  Pf: ✓? se piloto           │  Pf: ✓ (com 6 meses)        │
├─────────────────────────────┼─────────────────────────────┤
│  C: finanças (preto)        │  ★ DECISÃO FINAL            │
│  P0: ✗ custo                │  piloto 6 meses ✓           │
│  Pf: ✗ este ano             │  PENDENTE: aprovação diret. │
└─────────────────────────────┴─────────────────────────────┘
```

## Armadilhas e Erros Comuns

| Armadilha | Detalhe |
|-----------|---------|
| **Posição inicial ≠ posição final** | Nunca responda com base nos primeiros 30 segundos — posições mudam |
| **Concordância condicional ≠ concordância** | "Could work if X" não é "I support it" — a condição precisa ser registrada |
| **Misturar quem disse o quê** | Com 3 falantes é fácil confundir — o quadrante previne isso |
| **Ignorar o falante mais quieto** | O terceiro falante muitas vezes diz algo crucial que cai em questão |
| **Ficar fixo em um rosto** | Mova os olhos — expressões dos outros falantes durante uma fala também contam |
| **Escrever frases** | Se você escreve sentenças, perde o vídeo e fica atrasado — só símbolos e palavras-chave |

---

## Estratégia de Resposta

1. Nos **primeiros 5 segundos**: atribua A/B/C aos falantes e anote o identificador visual de cada um
2. Abra o quadrante no papel **antes** do áudio — estrutura pronta para preencher
3. Para cada falante: rastreie P0 → mudança → Pf (não só o momento inicial)
4. Ao ouvir uma condição ("as long as", "provided that"), anote com → no campo "condição"
5. Após o vídeo: responda primeiro as questões visuais (roupa, cena) enquanto a memória ainda está fresca
6. Preste atenção no **resumo final do moderador** — contém a decisão e o que ficou pendente
